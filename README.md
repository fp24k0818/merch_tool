# 物販チェックツール（CSV対応）

CSVファイルを入力することで、  
物販商品の **利益・利益率・在庫状況を自動判定** するPythonツールです。

スプレッドシート（Googleスプレッドシート / Excel）で管理している商品データを  
そのままCSVで取り込み、  
問題のある商品を一目で把握できるようにしています。

---

## 🔧 主な機能

- 利益額の自動計算
- 利益率（%）の自動計算
- 在庫なし商品の検出
- 利益率が基準未満の商品を自動でNG判定
- 判定理由（赤字 / 在庫なし / 低利益率）を明示
- 全件結果CSV / 問題商品CSV の自動出力
- 設定値（利益率基準）をCSVで変更可能
- Macでダブルクリック実行対応（run.command）

## 📝 入力CSV（input.csv）の形式

以下の列名でCSVを作成してください。

```csv
商品名,仕入価格,販売価格,在庫数
商品A,1000,1500,5
商品B,2000,1800,3
商品C,800,1200,0
Googleスプレッドシート / Excel で作成可能

UTF-8（CSV）形式で保存してください

⚙️ 設定ファイル（config.csv）
利益率の基準をCSVで変更できます。

key,value
low_profit_rate,40
上記例では「利益率40%未満」をNG判定します

数値を変更するだけで基準を調整できます

▶ 使い方（Mac）
方法①：ダブルクリック実行（おすすめ）
input.csv を main.py と同じフォルダに置く

run.command をダブルクリック

結果CSVが自動生成されます

方法②：ターミナル実行
python main.py
📤 出力ファイル
checked_items_YYYY-MM-DD.csv
→ 全商品の判定結果

problem_items_YYYY-MM-DD.csv
→ NG商品のみ抽出したCSV
